HTML Cheat Sheet

Element Description MDN Docs Red
Lists The <li> Tag is used to generate structured or unstructred lists within a HTML DOCTYPE. < OL > Is used for prioritizing list items in order of 1 to 10 for example. MDN LI Reference
Tables The <Table> Tag enables an element to be populated with Rows <tr> and Column <th> Values. This can enable an HTML page to show case data in a "Speadsheet" like manner or databasing. MDN Reference
Strong & Em If you need to emphasise text in Bold <Srtong> or Italics <Em> MDR Reference
DOCTYPE The Basis of identifying a text files purpose wether for HTML, CSS, JavaScript etc. MDN Reference
Footing Conceptually similar to how a Nav would function but more of a means to identify metadata / important information about the website. You would likely see <nav> and <li> be used in these cases. MDN Reference
